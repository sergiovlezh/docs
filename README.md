# Docs - Documents Manager

A self-hosted document management system inspired by [Paperless NGX](https://docs.paperless-ngx.com/).

## Description

The application allows you to **manage digital documents**, including:

- Loading documents with one or multiple files.
- Downloading associated files to a document.
- Organization through:
  - Tags
  - Descriptions
  - Notes
- Strict ownership isolation per user (ownership).
- Advanced operations such as **document fusion**.
